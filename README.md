# drymartina.se

Första versionen av webbplatsen och den digitala båtpärmen för Saga 27 **Dry Martina**.

## Filer

- `index.html` – sidans innehåll
- `style.css` – utseende och mobilanpassning
- `script.js` – mobilmeny och checklistor
- `bilder/` – foton från båten
- `dokument/` – PDF-manual och andra dokument

## Publicering

Projektet är avsett att publiceras med Cloudflare Pages från GitHub.

Framework preset: `None`  
Build command: lämnas tomt  
Build output directory: `/`
